Introduction & Basics

A powerful Python library for data manipulation and analysis

What is Pandas?

Pandas is an open-source Python library that provides high-performance, easy-to-use data structures and data analysis tools.

Key Features:

  • Fast and efficient DataFrame object for data manipulation
  • Tools for reading and writing data between in-memory data structures and different file formats
  • Data alignment and integrated handling of missing data
  • Reshaping and pivoting of datasets
  • Label-based slicing, indexing, and subsetting of large datasets
  • Time series functionality
# Installing Pandas
pip install pandas

# Importing Pandas
import pandas as pd

Core Data Structures

Series

1-dimensional labeled array

Can hold any data type

DataFrame

2-dimensional labeled data structure

Like a spreadsheet or SQL table

DataFrame Structure

Index Name Age City
0 Alice 25 New York
1 Bob 30 Paris
2 Charlie 35 London

Creating DataFrames

From Dictionary:

import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Paris', 'London']
}

df = pd.DataFrame(data)

From CSV File:

df = pd.read_csv('data.csv')

From Excel File:

df = pd.read_excel('data.xlsx')

Basic DataFrame Operations

# View first 5 rows
df.head()

# View last 5 rows
df.tail()

# Get DataFrame info
df.info()

# Statistical summary
df.describe()

df.columns

df.shape

Data Selection & Indexing

Selecting Columns:

# Single column
df['Name']

# Multiple columns
df[['Name', 'Age']]

Selecting Rows:

# By position (iloc)
df.iloc[0]  # First row
df.iloc[0:3]  # First 3 rows

# By label (loc)
df.loc[0]  # Row with index 0
df.loc[0:2, 'Name']  # Specific rows and columns

Filtering:

# Filter rows where Age > 25
df[df['Age'] > 25]

Data Analysis Example

Sample dataset visualization showing age distribution:

# Creating a bar plot
df['Age'].plot(kind='bar', title='Age Distribution')

# or using matplotlib
import matplotlib.pyplot as plt
df.plot(x='Name', y='Age', kind='bar')

Common Data Operations

Adding/Removing Columns:

# Add new column
df['Country'] = ['USA', 'France', 'UK']

# Drop column
df.drop('Country', axis=1, inplace=True)

Handling Missing Data:

# Check for missing values
df.isnull()

# Drop rows with missing values
df.dropna()

# Fill missing values
df.fillna(0)

Sorting:

# Sort by column
df.sort_values('Age', ascending=False)

Grouping & Aggregation

# Group by column and calculate mean
df.groupby('City')['Age'].mean()

# Multiple aggregations
df.groupby('City').agg({
    'Age': ['mean', 'min', 'max'],
    'Name': 'count'
})
